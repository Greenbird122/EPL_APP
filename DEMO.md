# REPAIR-AI — 3-minute demo script

Full platform: [https://repairai.co.ke/](https://repairai.co.ke/)

## Before you present

- Run `flutter pub get` and `flutter run` on a charged device or emulator.
- Optional: switch to **Kiswahili (SW)** on onboarding, login, or home.
- For voice demo on triage: grant microphone permission when prompted.

## Tap order

| Step | Screen | Action |
|------|--------|--------|
| 0 | Splash | Motherly quotes (~5s) while app loads |
| 1 | Onboarding | Swipe slides; toggle EN/SW and dark mode |
| 1b | Onboarding (last) | Optional **See how it works** |
| 2 | How it works | 4 steps; **Learn more** opens website (only external link in flow) |
| 3 | Login | Accept **Terms & Conditions**; **Try demo** → **Login** |
| 4 | Welcome | Quotes + smooth transition to home |
| 5 | Home | Hero, report symptoms card, WhatsApp 24/7 chat preview |
| 6 | Triage | Select **Vaginal Bleeding** + **Severe Abdominal Pain**; optional mic for notes |
| 7 | Analyzing | Steps + rotating motherly quotes (min 5s) |
| 8 | Result | Localized risk, **Why this result?**, **Call emergency** if high |
| 9 | Referral | Map image tap → Maps; **Send Referral** |
| 10 | My Reports | Open saved entry |
| 11 | Profile | Appearance, language, logout |

## Talking points

- Aligns with [repairai.co.ke](https://repairai.co.ke/) patient journey (register → symptoms → AI → referral).
- Mobile demo uses **on-device rules** with explainable output in EN or SW.
- Speech-to-text on symptom notes; WhatsApp for 24/7 human support.
- Demo disclaimer on home, triage, referral, and results—not a medical diagnosis.
